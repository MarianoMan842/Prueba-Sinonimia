The code is in spanish because the public of the game are students from Spain that the next year are gonna enter the university.

Este es un juego de sinónimos y antónimos para prácticar para el examen de PAU de lengua castellana. 
En selectividad hay una pregunta en la que tienes que definir sinonimia, antonimia, hiponimia, hiperonimia o polisemia y te piden ejemplos de una palabra.
El próposito de este juego es ayudar a los alumnos a realizar de la mejor forma esta pregunta del examen, claro que solo he hecho la parte de sinonimia y antonimia.
Te pieden que digas un sinónimo o un antónimo de una palabra para que practiques y aprendas, solo te dice si está bien o mal, no te da posibles respuestas, si quieres verlas pues vas al código o las buscas en Google o en el diccionario.
